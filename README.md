# Copa Ropa Deportiva - ERP/CRM Dashboard

Sistema ERP/CRM para la fábrica textil **Copa Ropa Deportiva**. Permite gestionar clientes (CRM), movimientos de inventario (Kardex) y crear/descargar cotizaciones y pedidos en PDF con renders de diseño.

## Características

- **Panel Principal (Dashboard):** Indicadores de ventas, órdenes activas, alertas de stock mínimo y clientes inactivos.
- **Cotizaciones y Pedidos:** Generador de documentos de cotización imprimibles en tiempo real con los logos correctos de la empresa.
- **Inventario (Kardex):** Registro de entradas y salidas de mercancía con búsqueda y filtros rápidos.
- **Clientes (CRM):** Directorio de contactos y alerta de inactividad para seguimiento comercial.

## Tecnologías Utilizadas

- **Framework:** Next.js (con TailwindCSS)
- **Componentes:** Shadcn UI & Lucide Icons
- **Lenguaje:** TypeScript

## Instalación y Desarrollo

1. Instalar dependencias:
   ```bash
   pnpm install
   ```

2. Ejecutar el servidor de desarrollo:
   ```bash
   pnpm dev
   ```

3. Abrir [http://localhost:3000](http://localhost:3000) en el navegador.
